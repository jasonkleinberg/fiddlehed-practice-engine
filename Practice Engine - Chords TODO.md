## What to do

1. Open the source `.sib` file in Sibelius (in the `Practice Engine - Sibelius for Export` folder).
2. Add **chord symbols** above the staff — real chord symbols (Sibelius: Notations ▸ Chord Symbol, or Cmd+K), **not** text. Use standard, simple chords that fit the melody, key, and fiddle-tune style (mostly I–IV–V). Every measure or every couple of measures is plenty.
3. For the 3 "convert" tunes, the chords are already there as text — just re-enter them as proper chord symbols and delete the text.
4. Export as **MusicXML** (uncompressed `.musicxml`) and send the files back, keeping the same filenames.

**Definition of done:** each file opens in a MusicXML reader and shows chord symbols (harmony) that match the melody. Simple and correct beats fancy.
